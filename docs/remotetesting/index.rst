======================
Удалённое тестирование
======================

Удалённое тестирование представляет из себя подобие модульных тестов,
только производимых с использованием внешней сторонней системы. Данная
система может удалённо командовать SAN-ом на предмет создания дисковых
массивов, логических разделов, iSCSI целей, а также запускать требуемые
утилиты для создания нагрузки.
